Key Concepts

🔵What is K-Means?

An unsupervised algorithm that groups data into K clusters by minimizing intra-cluster distance. It iterates: assign points → recompute centroids → repeat until convergence.

📐Standardization

We apply StandardScaler before clustering so all features have mean=0 and std=1. This ensures no single feature dominates due to scale differences.

📊Elbow Method

Plot inertia (WCSS) vs K. The "elbow" — where improvement slows — indicates the optimal number of clusters. For Iris, K=3 is the clear bend.

✅Silhouette Score

Measures how well-separated clusters are. Ranges from -1 to 1 — higher is better. A score above 0.5 indicates reasonable cluster structure.

🌸Iris Dataset

150 flowers across 3 species (Setosa, Versicolor, Virginica). Four features: sepal length/width and petal length/width. A classic ML benchmark dataset.

🌐3D Visualization

Plotly renders an interactive 3D scatter using sepal length, sepal width, and petal length. Rotate the chart to explore cluster separation from different angles.
